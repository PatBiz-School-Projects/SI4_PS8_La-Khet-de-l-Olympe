Règles de l'Olympe

Détruisez le Pharaon adverse avant qu'il ne détruise le vôtre ! À la fin de chaque tour, votre Sphinx tire un rayon mortel qui rebondit sur les miroirs.

Les Artefacts

  • Pharaon : Le roi. S'il est touché par un laser, vous perdez la partie. Il n'a aucune défense.
  • Sphinx : Indestructible. Il tire le laser à la fin de votre tour. Il ne peut pas se déplacer, seulement pivoter.
  • Pyramide : Possède 1 miroir en diagonale. Dévie le laser à 90°. Vulnérable sur le dos.
  • Scarabée : Possède 2 miroirs. Indestructible. Peut échanger sa place avec le Sphinx ou le Pharaon !
  • Anubis : Le bouclier divin. Bloque le laser de face, mais vulnérable sur les côtés.

Votre Tour de Jeu

Choisissez UNE SEULE action :

  • Pivoter une de vos pièces de 90°.
  • Déplacer une pièce d'une case (sauf Sphinx et Pharaon).
  • Placer une Pyramide de votre réserve sur le plateau.
  • Échanger la position de votre Scarabée.
⚡ Dès que votre action est finie, le Sphinx tire ! Calculez bien la trajectoire...
